## Overview

This is an **Interactive Road Accident Analysis Dashboard** built using **Power BI**. The dashboard provides detailed insights into road accidents, including:

* **Total casualties** and breakdown by severity (fatal, serious, slight)
* **Casualty distribution** by vehicle type (car, bus, bike, van, etc.)
* **Casualty data by road type** (single carriageway, dual carriageway, roundabout, etc.)
* **Monthly trend** comparison of casualties for the current year (CY) vs. previous year (PY)
* **Casualty distribution by urban/rural areas** and light conditions (day/night)
* **Geographical map** showing accident locations

The dashboard is ideal for **data analysts, traffic authorities, and safety researchers** interested in improving road safety.
